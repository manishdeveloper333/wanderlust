# Wanderlust Project

The Wanderlust Project is a sample repository created as part of a peer-graded DevOps assignment.
It demonstrates the use of GitHub best practices such as licensing, documentation, and version control.

## Description
This project is used to practice Git and GitHub concepts including:
- Creating repositories
- Adding an open-source license
- Writing a README file
- Managing branches and pull requests

## Usage
This repository is for learning and demonstration purposes.
Files in this repository can be modified to practice Git workflows.
